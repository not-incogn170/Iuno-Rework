# Installation guide
### Requirements 
- NodeJs
- ffmpeg
### in Terminal
1. clone this repo
   ```bash
   git clone https://github.com/not-incogn170/Iuno-Rework.git
   ```
3. open the 'Iuno-Rework' folder
4. install the modules/dependency
   ```bash
   npm i
   ```
6. make .env file
   ```env
   PHONE_NUMBER=62xxx
   OWNER_PHONE_NUMBER=62XXX
   ```
7. start the bot
   ```bash
   node .
   ```
> [!NOTE]
> You'll asked to connect the number using pairing code
